# Daycare Site

This is a full-stack daycare site built with Next.js, TypeScript, Tailwind CSS, and Clerk for authentication. It uses Prisma for database management.

## Project Goal

To provide a comprehensive platform for managing daycare services, including child information, bookings, and parent accounts.

## Current Status

The project is under active development. While the core features are in place, there are ongoing efforts to resolve build and deployment issues.
